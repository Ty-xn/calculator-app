# calculator app

A simple browser-based calculator built with HTML, CSS, and inline JavaScript.

## Files

- [calculator.html](calculator.html) — UI structure and calculator logic
- [style.css](style.css) — calculator styling

## Features

- Digit input (`0–9`)
- Basic operations: addition, subtraction, multiplication, division
- `AC` clear function
- `=` result evaluation

## How to Run

1. Open [calculator.html](calculator.html) in a web browser.
2. Click the buttons to perform calculations.

## Notes

- This project uses inline event handlers in [calculator.html](calculator.html).
- Styles are defined in [style.css](style.css).
